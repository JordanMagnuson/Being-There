bolting   , unbolting   

unbolting something   (past tense unbolted)

The unbolting action (corresponding to UNLOCK SOMETHING) can be used in two ways. If the object to be unlocked has a known-key, and the known-key is available to the player, then UNLOCK WHATEVR with be redirected to UNLOCK WHATEVER WITH KNOWN KEY. If the object to be unlocked has a matching key of no-key, then it doesn't need a key to unlock it, so that UNLOCK WHATEVER will simply go ahead and unlock it.

Typed commands leading to this action

"unlock [something]" 

"unlock [door]" 

"unlock [locked door]" 

"unlock [container]" 

"unlock [locked container]"
