DESCRIPTION
------------------------------------------------------------------------

An extremely experimental little work of interactive fiction with pictures, 
about existence and Korea.

------------------------------------------------------------------------
PLAYING THE GAME
------------------------------------------------------------------------

Being There requires that you download and install a free Glulx interpreter.

For Windows: download WinGlulxe
For Mac OS X: download Zoom
For Linux: download Glulxe for Linux

Once you’ve installed a Glulx interpreter, run it, and use it to open
the "Being There.gblorb" file.
